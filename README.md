### 禁闻聚合安卓版

禁闻聚合包含三个版块：海外中文新闻媒体、翻墙必看视频、热门YouTube频道， 让您免翻墙即可自由浏览墙外诸多资源

下载安装 [禁闻聚合安卓版](https://raw.githubusercontent.com/gfw-breaker/bn-android/master/resource/banned-news.apk)，若有页面无法正常加载，请点击右上角的“刷新”
